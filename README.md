# בוחן תרגול — ניהול מערכות רכש ומלאי מתקדמות

אפליקציית תרגול למבחן בקורס **ניהול מערכות רכש ומלאי מתקדמות** (825560003, אלי צימלס, בר אילן).

- **231 שאלות**: 5 שאלות רשמיות של המרצה (עם מפתח תשובות) + 104 דגשים ודוגמאות מכל 10 הקלטות הזום + 84 שאלות תרגול + 38 שאלות בסגנון המבחן (ארוכות, "הפוך על הפוך", מרובות-נכונות), כולל כיסוי חומרי חובת-הקריאה (ניהול מלאי — ABC/פארטו; ניהול סיכונים מבוסס מידע) ופרטי חוק חובת המכרזים (העדפות, תנאי סף, סיווג עסק). כל 4 המסיחים בכל שאלה קרובים וסבירים (בוחן הבנת הנקרא), כולל שאלות "כל התשובות נכונות"/"א'+ב'"/"למעט" שחלקן מלכודת.
- פורמט זהה למבחן: 4 תשובות לשאלה, כולל תשובות מורכבות ("תשובות א'+ב' נכונות" / "כל התשובות נכונות").
- סינון לפי נושא, מפת שאלות, מונה ניקוד ורצף, ערבוב, מצב כהה/בהיר, וחישוב ציון.
- קובץ יחיד ועצמאי (`index.html`) — נפתח בכל דפדפן, ללא התקנה.
